The dataset for this project is available at the following link.
https://drive.google.com/drive/folders/1H0x0Z-_SY1LXogiedZ75fmsZEee8NHVv?usp=sharing
The dataset is in the form of a Google Drive folder. 

The folder contains the following directories:
- skinProblem
    - wrinkles
    - darkspots
    - blackheads
    - acne
    - skin_problem_dataset.zip
- skinCareProducts
    -products.csv

The skinProblem directory contains images of skin problems such as wrinkles, darkspots, blackheads, etc. 

The skinCareProducts directory contains csv file of skin care products such as face wash, moisturizer, sunscreen, etc.
